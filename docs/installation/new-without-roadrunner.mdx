# Fresh installation without roadrunner

Simply clone the `ftc17191/FTCLayer-quickstart` repository and start working!
Download from [here!](https://github.com/ftc17191/FTCLayer-quickstart/archive/refs/heads/master.zip)

## From the command line

> This is assuming you have Git installed, if you don't, (click here)[https://git-scm.com/book/en/v2/Getting-Started-Installing-Git] or use another installation method.
```bash
git clone https://github.com/ftc17191/FTCLayer-quickstart.git
```
