## What _is_ FTCLayer?

FTC Layer is an open source project designed to provide abstractions over the base FTC SDK. FTC Layer is designed to make programming easier, less time consuming, and simpler.

If you are really interested, check out the [github repo](https://github.com/ftc17191/FTCLayer)!
